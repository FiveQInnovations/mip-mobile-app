# Instructions for the Builder Agent

## Ticket Parameter
You will be given a Ticket as a parameter. This could be a file, a ticket number, or a description of the ticket. Use this parameter to find the right ticket in the `tickets` folder. Consult the tickets/README.md for an overview.

## Instructions
Put this ticket into In Progress. Commit.

Think through how to implement this ticket and make a todo list.
Then implement the todos. 

Rebuild in Release. 
Perform manual MCP exploration on the simulator following @mip-mobile-app/docs/mcp-simulator-exploration.md .
Run the ios Maestro tests and ensure they pass.

Move ticket to QA if all good.

Commit all relevant files when done.
